# Política de Privacidade — Project Phoenix Manager

**Última actualização:** 15 de Julho de 2026  
**Versão da app:** 0.8.9

## Resumo

O **Project Phoenix Manager** é um jogo de gestão desportiva offline-first. **Não recolhemos dados pessoais**, não usamos analytics de terceiros e não enviamos informação para servidores externos.

## Dados armazenados no dispositivo

| Dado | Onde | Finalidade |
|------|------|------------|
| Saves de carreira | SharedPreferences / UserDefaults local | Continuar o jogo |
| Preferências da app | Armazenamento local do SO | Funcionamento da app |

Os saves contêm estado do jogo (clubes, jogadores, calendário, finanças simuladas). **Não incluem nome, email, localização nem identificadores reais.**

## Dados que não recolhemos

- Conta de utilizador ou login
- Localização GPS
- Contactos, fotos ou ficheiros pessoais
- Identificadores de publicidade
- Telemetria para servidores externos

## Partilha com terceiros

Não vendemos, alugamos nem partilhamos dados com terceiros. Não há SDKs de analytics ou publicidade integrados nesta versão.

## Crianças

A app não se destina especificamente a menores de 13 anos. Não recolhemos dados de crianças de forma consciente.

## Apagar dados

- **Android:** Definições → Apps → Phoenix Manager → Armazenamento → Limpar dados  
- **macOS:** Remove saves ao apagar preferências da app ou reinstalar após limpar UserDefaults  
- **Dentro do jogo:** Menu carreira → apagar slot de save

## Alterações

Publicaremos alterações nesta página (ou URL equivalente na loja) quando a política mudar.
